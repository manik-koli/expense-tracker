# Expense Tracker

A simple command-line expense tracker built with Python.

## Features
- Add daily expenses with date, amount, category and description
- View monthly expenses with total
- View yearly summary month by month
- Data saved in CSV file

## How to Run
1. Make sure Python is installed
2. Run this command:
```
python expense_tracker.py
```

## Categories
- Food
- Travel
- Rent
- Other

## Tools Used
- Python
- CSV module
